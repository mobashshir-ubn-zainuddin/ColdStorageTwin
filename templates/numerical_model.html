Numerical Model

Coupled Heat-Moisture Transfer

Cold storage involves more than just heat transfer. The complete physics includes:

  • Heat diffusion — thermal energy spreads through the domain
  • Moisture diffusion — water vapor migrates through the air
  • Latent heat effects — condensation and frost formation release energy
  • Temperature-humidity coupling — temperature and moisture fields are interdependent

This requires solving two coupled PDEs simultaneously.

Governing Equations
1. Energy Equation (Temperature)

The thermal behavior includes heat diffusion plus latent heat release from moisture changes:

$$\rho C_p \frac{\partial T}{\partial t} = k \nabla^2 T + L_v \frac{\partial w}{\partial t}$$

Where:
  • \(T(x, y, z, t)\) = Temperature field (°C)
  • \(w(x, y, z, t)\) = Moisture content (fraction)
  • \(k\) = Thermal conductivity (W/(m·K))
  • \(\rho\) = Density (kg/m³)
  • \(C_p\) = Specific heat capacity (J/(kg·K))
  • \(L_v\) = Latent heat of vaporization (J/kg) ≈ 2.26×10⁶ J/kg
  • \(\nabla^2 T\) = Laplacian operator
2. Moisture Diffusion Equation

Moisture behaves like a diffusing scalar:

$$\frac{\partial w}{\partial t} = D_m \nabla^2 w$$

Where:
  • \(w\) = Moisture concentration (mass fraction)
  • \(D_m\) = Moisture diffusivity (m²/s) ≈ 10⁻⁶ m²/s

Physical Coupling: When moisture changes (condensation or evaporation), latent heat is released or absorbed. This directly affects the temperature field. Conversely, temperature affects saturation moisture levels, creating a two-way coupling.

Heat-Moisture Coupling & Condensation
Coupling Flow

Moisture Diffusion → Condensation (if T < Tdew) → Latent Heat Release → Temperature Changes

Dew Point Condition

Condensation occurs when the temperature drops below the dew point:

$$T < T_{dew}$$

The dew point is calculated using the Magnus formula:

$$T_{dew} = \frac{b \cdot \gamma(T, RH)}{a - \gamma(T, RH)}$$

where \(\gamma(T, RH) = \frac{aT}{b + T} + \ln(RH)\)

Magnus Constants:
  • \(a = 17.27\)
  • \(b = 237.7\) °C
  • \(RH\) = Relative humidity (0–1)
Numerical Solution Method

Both equations are solved using the Explicit Finite Difference Method (EFDM) with a staggered update scheme.

Spatial Discretization

The domain is divided into a structured rectangular grid with uniform spacing:

$$\Delta x = \frac{L_x}{n_x - 1}, \quad \Delta y = \frac{L_y}{n_y - 1}, \quad \Delta z = \frac{L_z}{n_z - 1}$$

Discrete Update Formulas
Moisture Update (First)

$$w_{i,j,k}^{n+1} = w_{i,j,k}^n + r_w \left[ \nabla^2 w \right]$$

Where \(r_w = \frac{D_m \Delta t}{(\Delta x)^2}\)

Temperature Update (Second, with Latent Heat Coupling)

$$T_{i,j,k}^{n+1} = T_{i,j,k}^n + r_T \left[ \nabla^2 T \right] + \frac{L_v}{\rho C_p} \left( w_{i,j,k}^{n+1} - w_{i,j,k}^n \right)$$

Where \(r_T = \frac{\alpha \Delta t}{(\Delta x)^2}\) and \(\alpha = \frac{k}{\rho C_p}\)

Laplacian Expansion (3D)

$$\nabla^2 T = T_{i+1,j,k} + T_{i-1,j,k} + T_{i,j+1,k} + T_{i,j-1,k} + T_{i,j,k+1} + T_{i,j,k-1} - 6T_{i,j,k}$$

Stability Analysis

The explicit method is conditionally stable. For the coupled system, both stability conditions must be satisfied.

Stability Criteria

For the 3D coupled heat-moisture system with uniform grid spacing:

$$r_T = \frac{\alpha \Delta t}{(\Delta x)^2} \leq \frac{1}{6}$$

$$r_W = \frac{D_m \Delta t}{(\Delta x)^2} \leq \frac{1}{6}$$

Important: Both \(r_T\) and \(r_W\) must satisfy the CFL condition. If either is violated, the numerical solution will oscillate and diverge. This application automatically checks both stability conditions.

Stability Implications
Parameter Effect on Stability
Thermal diffusivity (α) Lower α allows larger Δt
Moisture diffusivity (Dm) Lower Dm allows larger Δt
Grid spacing (Δx) Larger Δx allows larger Δt (proportional to Δx²)
Typical constraint Usually Dm > α, so moisture stability is more restrictive
Boundary Conditions

Both temperature and moisture require boundary conditions on all faces of the domain. This implementation uses Dirichlet boundary conditions (fixed values):

Temperature Boundaries

$$T_{\text{walls}} = T_{\text{wall}}, \quad T_{\text{inlet}} = T_{\text{inlet}}$$

Moisture Boundaries

$$w_{\text{walls}} = w_{\text{wall}}, \quad w_{\text{inlet}} = w_{\text{inlet}}$$

Initial Conditions

$$T(x, y, z, t=0) = T_{\text{initial}}, \quad w(x, y, z, t=0) = w_{\text{initial}}$$

Implementation Algorithm
Time Stepping Loop (Coupled System)
for each time step n = 0 to N_steps:
    # Check stability for BOTH fields
    if r_T > 1/6 or r_W > 1/6:
        raise StabilityError("Configuration is unstable")
    
    # STEP 1: Update moisture field (diffusion only)
    for interior points (i,j,k):
        W_new[i,j,k] = W[i,j,k] + r_W * (
            W[i+1,j,k] + W[i-1,j,k] +
            W[i,j+1,k] + W[i,j-1,k] +
            W[i,j,k+1] + W[i,j,k-1] - 6*W[i,j,k]
        )
    
    # STEP 2: Update temperature (diffusion + latent heat)
    for interior points (i,j,k):
        laplacian_T = (T[i+1,j,k] + T[i-1,j,k] +
                       T[i,j+1,k] + T[i,j-1,k] +
                       T[i,j,k+1] + T[i,j,k-1] - 6*T[i,j,k])
        
        T_new[i,j,k] = T[i,j,k] + r_T * laplacian_T
                     + (Lv / (ρ*Cp)) * (W_new[i,j,k] - W[i,j,k])
    
    # STEP 3: Apply condensation if moisture exceeds saturation
    W_sat = calculate_saturation_moisture(T_new)
    where W_new > W_sat:
        excess = W_new - W_sat
        W_new = W_sat
        T_new += (Lv / (ρ*Cp)) * excess  # Latent heat release
    
    # STEP 4: Apply boundary conditions
    T_new[boundaries] = T_wall or T_inlet
    W_new[boundaries] = W_wall or W_inlet
    
    # Advance to next time step
    T, W = T_new, W_new

Accuracy Considerations
  • Spatial Order: Central differences have second-order accuracy: O(Δx²)
  • Temporal Order: Forward Euler has first-order accuracy: O(Δt)
  • Coupling: Staggered update (moisture first, then temperature) maintains stability
  • Conservation: Condensation model conserves total energy (latent + sensible)
  • Grid Refinement: Smaller Δx requires Δt ∝ Δx² for stability
